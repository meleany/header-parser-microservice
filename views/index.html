Request Header Parser Microservice

FreeCodeCamp APIs & Microservices Project

The Request Header Parser App can get the IP address, preferred languages and system infos from the user's device. To use this app, append api/whoami at the end of the url. In other words, the API endpoint is:
GET [project_url]/api/whoami

  1. The preferred language is obtained from the header Accept-Language.
  2. The system info is extracted from the header User-Agent.

Example of Usage:

https://headerparsermel.glitch.me/api/whoami

Example Response:

{"ipaddress":"159.20.14.100","language":"en-US,en;q=0.5", "software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
